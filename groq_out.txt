QUESTION: If f(x) = 2x^2 + 3x - 1, what is the vertex of the parabola represented by this function?
A) (-3/4, -13/8)
B) (-3/2, -7/2)
C) (1/2, 3/2)
D) (-3/4, -7/8)
ANSWER: A) 
EXPLANATION: Betam tiru, to find the vertex, we need to use the formula x = -b / 2a, where a = 2 and b = 3 in this case, so x = -3 / (2*2) = -3/4. Then, substitute this x-value back into the function to find the corresponding y-value, y = 2(-3/4)^2 + 3(-3/4) - 1 = 2(9/16) - 9/4 - 1 = 9/8 - 9/4 - 1 = -13/8.
